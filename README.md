# MoodReset Privacy Policy

This repository hosts the Privacy Policy for the MoodReset iOS app.

## Privacy Policy URL

The privacy policy is available at:
**https://aglawns.github.io/moodreset-privacy/**

This URL should be used in App Store Connect when submitting the MoodReset app.

## Files

- `PRIVACY_POLICY.md` - Markdown version of the privacy policy
- `index.html` - HTML version for GitHub Pages

## Updates

To update the privacy policy:
1. Edit `PRIVACY_POLICY.md` or `index.html`
2. Commit and push changes
3. GitHub Pages will automatically update
